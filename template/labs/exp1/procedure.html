Earthquake Time Series Analyser

Procedure

1. Data Input and Setup

Begin by loading your seismic data using one of these methods:

  • Click the Upload button to select a CSV file
  • Drag and drop your CSV file into the designated area
  • Select a preloaded sample dataset from the dropdown menu

Once loaded, enter the sampling frequency (in Hz) for your data to ensure accurate analysis.

2. Filter Selection and Configuration

The tool offers several filtering options to enhance signal quality:

  • Low-Pass Filter: Removes high-frequency components while preserving low frequencies
  • High-Pass Filter: Eliminates low-frequency noise while retaining high frequencies
  • Band-Pass Filter: Isolates a specific frequency range for detailed analysis

Adjust filter parameters using the provided controls:

  • Set cutoff frequencies using the slider controls
  • Adjust filter order to control the sharpness of frequency cutoffs
  • Use preset configurations for common seismic analysis scenarios

3. Visualization and Analysis

The interface provides two main visualizations:

  • Time Series Plot:
    • Displays amplitude variations over time
    • Shows both original and filtered signals
    • Enables direct comparison of signal changes
  • Spectrogram:
    • Visualizes frequency content evolution
    • Helps identify seismic phases and events
    • Shows energy distribution across frequencies

4. Interactive Features

Utilize these interactive features for detailed analysis:

  • Zoom: Use mouse wheel or pinch gestures
  • Pan: Click and drag across plots
  • Data inspection: Hover over points for exact values
  • Reset view: Return to full data view

5. Parameter Optimization

Fine-tune your analysis by adjusting:

  • Filter order for processing precision
  • Spectrogram overlap percentage for resolution control
  • Window size for time-frequency trade-offs
  • Color scaling for better visualization

6. Results and Export

After completing your analysis:

  • Review the filtered signal quality
  • Compare original and processed data
  • Download processed data as CSV file
  • Save visualization settings for future use
